1. Открыть разметку
2. Сохранить разметку как...

+4. Экспортировать разметку
+5. Очистить фрейм
+6. Сортировать кадры
+7. Воспроизвести
8. Использовать сопровождение
9. Нстроить параметры

=============
Актуальные задачи.

1. Работа с несколькими файлами разметки, связанными с одним видеороликом.
        1.1. реализовать текущее имя файла разметки.
        1.2. реализовать операции Create, Open и Save As
2. Работа в режиме воспроизведения разметки, там возможны баги.
        2.1 Перенести воспроизведение разметки в функцию TPhVideoMarkTool::Playback
3. Редактор ярлыков.
        3.1 Использовать функцию TPhVideoMarkTool::LabelStatistics
4. Сохранять информацию о bw и bh в разметке.
5. Использовать возможности TLFDetectedItem для сохранения ID объекта. Реализовать
        для некоторых объектов установку ID. По умолчанию -1. Все остальные ID должны
        приводить к генерации нового UUID и возможности их ассоциирования с
        текущим объектом.
        5.1. Визуализация списка ID - UUID
        5.2. Визуализация ID в ListView2
6. Использование алгоритмов сопровождения для разметки.
7. Реализция Popup для ListView1
