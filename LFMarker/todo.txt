Database commands todo:
1. Create database
        создает базу данных в случае, если в текущем каталоге нет файла Dictionary.xml
        для создаения вызывает диалог редактора словаря.
2. Edit dictionary
        вызывает диалог редактора словаря.
        2.1 Добавить слово в словарь
        2.2 Удалить слово из словаря
        2.3 Изменить параметры сканера. (минимальный размер, базовую высоту, базовую ширину,
        шаг по масштабу, шаг по пространству)
        2.4 Изменить слово в словаре.
3. Copy database
        Копирует базу данных в указанный каталог.
4. Clear database
        Удаляет всю разметку и файл словаря.
6. Convert database
        Преобразует данные в соответствии с установками пользователя.
7. Split database
        - удаляется из набора команд (относится к системе построения детектора)
8. Do analysis
        - выполняет автоматическую разметку базы данных
9. Export image fragments -
        - удаляется из набора команд и переносится к системе построения детектора
