Проект Optophotonics data processor

назначение:

1. Визуализация оптоакустических данных
2. Разметка данных
3. Подготовка данных для обучения
4. Автоматическая сегментация данных c использованием Python

============

+1. Конвертация tiff файла в набор awp изображений
+2. навигация по awp изображениям
+3. Пороговая бинаризация изображения и визуализации бинарных изображений
+4. Выделение контуров и сохранение их в TLFObjectList
+5. Визуализация контуров на бинарном изображении и в трехмерном пространстве
6. Кластеризация контуров и сохранение кластеров на диск.
7. Визуализация кластеров.
8. Установка параметров алгоритма из пользовательского интерфейса.

=======
Алгорим анализа оптоакустических изображений.

1. Оптоакустическое изображение представляет собой трехмерную матрицу
с которой каждый слой содержит интенсивность сигнала на заданной глубине сканирования.


=======
Параметры алгоритма.

1. Радиус ядра сглаживания.
2. Порог бинаризации
