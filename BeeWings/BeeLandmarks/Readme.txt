База данных.
===========
База данных представляет собой два xml файла beelandmarks0.xml и
beelandmarks.xml.

beelandmarks0.xml - результат анализа пробы с использованием алгоритма
распознавания особых точек.
beelandmarks.xml - результат редактирования местоположениея особых точек
с использованием визуального редакотора или с использованием импорта TPS файла.


Импорт TPS файлов.
==================
1. Импорт производится в текущую папку, заданную значением переменной
DirectoryListBox1->Directory

2. Если в этой папке находится база данных, то в ней заменяется файл
beelandmarks.xml

3. Если в этой папке отсутствует база данных, то в ней создается файл
beelandmarks.xml и, в зависимости от установленных опций настройки
либо заменяется файл beelandmarks0.xml, либо выполняется анализ пробы
для создания файла beelandmarks0.xml

4. При импорте TPS производится проверка наличия файла изображения
в текущей папке.
